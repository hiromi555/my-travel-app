# ✈️ 旅のしおりアプリ (Travel Itinerary App)

自分専用の旅行計画を作成・管理できるWebアプリケーションです。
「オフラインでもデータ移行が可能」「スマホで見やすいUI」。


## ✨ 特徴
- **完全サーバーレス:** データベースを使わず、ブラウザだけで完結します。
- **オフラインデータ移行:** QRコードを使って、PCで入力したデータをスマホに簡単に転送できます。
- **スマホ最適化:** アコーディオンUIやタブ切り替えを採用し、移動中でも見やすいデザインです。

## 🛠 技術スタック
- React
- Vite
- qrcode.react (QRコード生成)
- lz-string (データ圧縮)

## ⚠️ 注意事項・仕様について

### 1. データの保存について (localStorage)
このアプリは、入力されたデータを**ブラウザのローカルストレージ (localStorage)** に保存します。
- サーバーには一切データ送信されません（プライバシー面でも安心です）。
- **注意:** ブラウザの **「履歴」や「Cookie・サイトデータ」** を削除したり、シークレットモードで利用してタブを閉じたりすると、保存データは消えてしまいます。

### 2. QRコードでのデータ転送について
PCからスマホへデータを送る際、QRコードに情報を圧縮して埋め込んでいます。
- **制限:** Googleマップなどの「非常に長いURL」や、大量の長文メモが含まれる場合、QRコードが細かくなりすぎて読み取れないことがあります。
- **対策:** 読み取れない場合は、URLを短縮するか、データを少し減らしてから再度お試しください。
